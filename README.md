# Análisis de señales en el dominio del tiempo y la frecuencia

## Descripción
Este repositorio contiene el desarrollo de una práctica académica cuyo propósito es analizar señales en el dominio del tiempo y en el dominio de la frecuencia utilizando la Transformada de Fourier. Para ello, se emplea el lenguaje de programación Python junto con librerías especializadas para el procesamiento y la visualización de señales.

## Objetivo
Analizar una señal compuesta en el dominio del tiempo y aplicar la Transformada de Fourier para identificar las frecuencias que la conforman, permitiendo una mejor comprensión del comportamiento de las señales y su representación en distintos dominios.

## Metodología
1. Se definió un eje de tiempo con una frecuencia de muestreo adecuada.
2. Se generaron señales senoidales de distintas frecuencias.
3. Se combinó la señal para obtener una señal compuesta.
4. Se representó la señal en el dominio del tiempo mediante una gráfica.
5. Se aplicó la Transformada de Fourier utilizando la función FFT.
6. Se visualizó el espectro de frecuencia para identificar las componentes de la señal.

## Tecnologías utilizadas
- Python
- NumPy
- Matplotlib
- Visual Studio Code

El análisis realizado demuestra que la Transformada de Fourier es una herramienta fundamental para el estudio de señales, ya que permite identificar sus componentes frecuenciales a partir de su representación temporal, facilitando su análisis y comprensión.
